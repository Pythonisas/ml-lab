* ML1 — Predicción de precios de vivienda (regresión)

Este lab construye un mini-sistema reproducible que estima el *precio* de una vivienda a partir de características simples.

- Trabajo: *tríos*
- Ejecutable: *terminal* (scripts Python)

** Estructura
#+begin_example
ml-vivienda-ml1/
├─ data/
│  ├─ README.md
│  └─ viviendas.csv            (opcional: dataset de ejemplo)
├─ docs/
│  ├─ practica-ml-precio-vivienda.org
│  ├─ mi-precio-vivienda_ML1.md
│  └─ conseguir-dataset-CAM.md
├─ models/                     (se crea al entrenar)
├─ src/
│  ├─ generate_dataset.py
│  ├─ train_incompleto.py
│  ├─ predict_incompleto.py
│  ├─ features.py
│  └─ utils.py
└─ requirements.txt
#+end_example

** Instalación
#+begin_src bash
cd ml-vivienda-ml1
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
#+end_src

** Generar dataset (sintético)
#+begin_src bash
python3 src/generate_dataset.py --out data/viviendas.csv --n 300 --seed 42
#+end_src

** Entrenar
#+begin_src bash
python3 src/train_incompleto.py --data data/viviendas.csv --save
#+end_src

** Predecir (CLI)
#+begin_src bash
python3 src/predict_incompleto.py --m2 75 --habitaciones 3 --banos 2 --zona centro --antiguedad 10
#+end_src

Si no pasas argumentos, el script preguntará por pantalla.
